# Iconos de Frida

Coloca aquí los iconos de la aplicación en los siguientes formatos y nombres exactos:

1. `icon.png` (512x512 px) - Para desarrollo y Linux.
2. `icon.ico` - Para compilar en Windows.
3. `icon.icns` - Para compilar en macOS.

Estos archivos serán detectados automáticamente por Electron y empaquetados por `electron-builder`.
